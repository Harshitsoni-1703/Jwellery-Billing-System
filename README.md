✨ Satyam Jewellers - Luxury Management System
Satyam Jewellers is a modern, web-based Billing and Khata (Ledger) Management System designed specifically for boutique jewelry stores. It combines a high-end aesthetic with powerful real-time data to streamline daily operations.

🚀 Key Features
Real-Time Market Data: Integrated with GoldAPI.io to fetch live global market prices for Gold (24K) and Silver (1kg), automatically converted to INR.

Multi-Item Invoicing: Add multiple ornaments (Rings, Chains, Bangles, etc.) within a single bill with independent weight and price tracking.

Customer Khata (Ledger): Maintains a complete database of customer details including Name, Mobile Number, Address, and Purchase Date.

Dynamic Balance Tracking: Automatically calculates the "Baki" (Remaining Balance) for each transaction to manage credit (Udhari) effectively.

Luxury Glassmorphism UI: A premium dark-themed interface with golden accents and smooth CSS animations for a high-end user experience.

Persistent Storage: Uses Browser LocalStorage to ensure all records remain safe even after closing or refreshing the tab.

🛠️ Tech Stack
Structure: HTML5 (Semantic Tags)

Styling: CSS3 (Custom Animations, Flexbox, Grid, Glassmorphism)

Logic: Vanilla JavaScript (ES6+, Fetch API, LocalStorage)

Data Feed: External API integration for live commodity pricing.
